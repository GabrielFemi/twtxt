About twtxt

twtxt is a twtxt client in the form of a web application. This is a hosted instance where you can signup to create an account and starting posting and following feeds of others, bots or news sources.

The timeline view displays the latest posts of users on this instance. Since twtxt itself is a federated and decentralised micro-blogging format there is no "global" timeline.

There is also a builtin /registry/v1 and /api/v1 that serves as a registry and search endpoint for keyword and mentions and direct API access to this instance.
